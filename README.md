# My Portfolio

This repository contains the full-stack portfolio project.

## Project Structure

- `frontend/` — React + Vite + Tailwind CSS portfolio frontend
- `backend/` — Laravel backend with Docker setup

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS
- GSAP

Backend:
- Laravel
- Docker

## Environment

- Backend runs in Docker
- Nginx runs in Docker
- Database runs on local machine

## Notes

The local database is not included in this repository.
Environment files such as `.env` are ignored and should be created locally.

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
